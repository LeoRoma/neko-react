# Neko-react

## Description
- Website built using the ```thecatapi```.
- The website is able to gain information of the cat breed.
- Still is underconstruction.

## Instructions
- Open your terminal.
- Clone the project typing: ```https://github.com/LeoRoma/neko-react.git```.
- Enter in folder project with ```cd neko-react``` and ```cd neko```.
- run ```npm install``` to install dependencies.

## Tech
- React
- React Router Dom
- MDBoostrap
- Material UI
- react-bootstrap bootstrap
- ReactPlayer
